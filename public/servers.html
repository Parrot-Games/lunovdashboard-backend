Server Management FAQ

Common questions about managing Lunov on your servers

How do I add Lunov to my server?
Click the "Invite Bot" button on any server card or use the invite link from the homepage. You need "Manage Server" permissions to add the bot.
Can I customize the prefix for my server?
Yes! Use the command "ln!prefix [new prefix]" to change it. You can also configure it through the server management dashboard.
How do I set up moderation features?
Access the moderation settings through the server management panel. You can configure automod, set up logging channels, and customize moderation rules.
Why can't I see all my servers?
You need to have "Manage Server" permissions in a server for it to appear here. Lunov must also be present in the server.
How do I remove Lunov from a server?
You can kick Lunov like any other member from your server. Alternatively, server owners can go to Server Settings > Members and remove the bot.
Can I transfer server ownership?
Server ownership transfer is handled through Discord's native system. Lunov settings will remain configured after ownership transfer.
